# RPG-Game
-------------------
A prototype RPG game created in GameMaker Studio 1.4. The prototype has two control options, a gamepad and a keyboard.

# Controls
-------------------
Control on keyboard:

* Movement: Arrow Up, Down, Left, Right
* Attackc: X
* Dash: C

You can also play this game on a gamepad from xbox.
